## ✅ 1. 확정됨 (재논의 X)
- **제품**: 담당자별 AI 협업 콘솔 — 로그인 → 본인 역할 대시보드. (7/2·7/3)
- **UX(차별화)**: 키보드-퍼스트, 스페이스 2회 승인, **"에이전트가 일하고 사람은 눈만"**. (7/2·7/3)
- **에이전트 구조**: 무기 = 스킬 + MCP + 은행 DB + 국가 오픈 API + LLM. **최종 승인·책임 = 사람, 고객 PII 비반출**, 케이스 메모리. (7/2·7/3)
- **D1 계열사 = 전북은행 + JB우리캐피탈 2개**. (7/3)
- **D4 구동 = 하이브리드** — 클로드/코덱스 연결 + 로컬 모델 실동작 시연, 최종 테스트 = 이승보 PC. (7/3)
- **히어로 = 전주 카페 대출심사** / **발표자 = 김주용·김민주**(둘 다 대비). (7/2·7/3)
- **디자인 시스템·UX 라이팅 1차 완료**. (7/2)
- **발표/평가 구조**(공식 [[본선_안내문]]): 예선 5분+2분 Q&A / 결선(상위5팀) 15분+5분 Q&A · **평가 5축×20 + 시연영상 가점5 = 105점** · 발표 스킬 항목 없음(내용 승부).
- **역할 분담**: 이승보=설계·개발 / 김민주=시연·발표·브랜딩 / 김주용=문서·API·시각화 / 최영욱=서포트.
